LINKS EXTERNOS

A TAG DE LINKS É <a> que significa âncora e dentro dela vai escrever o texto que se quer, para o usuário clicar. 
Exemplo: Clique aqui   

Mas precisa colocar o atributo href="" 
Este atributo é que "valida" o link, através da URL,no código que está sendo escrito. 

Para o usuário não saia do nosso site e esse link abra em uma outra aba, inserimos o atributo _target=""_blank    na TAG <a>.

Geralmente usamos o _blank quando queremos que o link leve para uma outra página. Quando tudo for ficar na mesma página, não precisa usar essa propriedade.

O atributo rel no HTML é como uma etiqueta que diz ao navegador qual é a relação entre a página atual e a página para a qual o link está apontando. Por exemplo, se o link for para a página de um autor, você pode usar rel="author" para informar ao navegador que o link é para a página do autor

o atributo target é usado para especificar onde abrir o link. Pra especificar o endereço do link, você deve usar o atributo href. Por exemplo: Clique aqui.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Quinto exercício do módulo de HTML básico do curso DevQuest.">
  <title>Links Externos</title>
</head>
<body>
  <h1>Links externos</h1>
  <p>Esta página contém um exemplo de link externo, o qual leva para outro website.</p>
  <a href="https://www.youtube.com/watch?v=-5CdAup0o-I" target="_blank">Clique aqui para ver um vídeo de gatinho!</a>
</body>
</html>

Listas

As listas são elementos HTML extremamente importantes na hora de criar um site, basicamente em todos os sites que abrir é possível ver uma lista de elementos. Nessa aula, serão vistas 2 tipos de listas: as ordenadas e não ordenadas

​Link de tipos de listas no HTML5
https://www.w3schools.com/html/html_lists.asp

Lista NÂO ordenada:  Lista que não tem ordem

TAG <ul>  significa Un order list  ( lista não ordenada)
Para cada item da lista será criado a TAG <li> que significa list item.

 



<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Oitavo exercício do módulo de HTML básico do curso DevQuest." />
  <title>Listas</title>
</head>
<body>
  <h1>Listas</h1>
  <p>Esta página contém um exemplo de lista não ordenada e um exemplo de lista ordenada.</p>
  <h2>Lista de compras (não ordenada)</h2>
  <ul>
    <li>Pão</li>
    <li>Tomate</li>
    <li>Queijo</li>
    <li>Cenoura</li>
  </ul>
  <h2>Lista de aprovados (ordenada)</h2>
  <ol>
    <li>Ana</li>
    <li>João</li>
    <li>Mariana</li>
  </ol>
</body>
</html>
